# anaypatil.me

My personal portfolio and blog — built with Next.js, Tailwind CSS, and MDX.

## Stack

- **Framework** — [Next.js 14](https://nextjs.org) (App Router)
- **Styling** — [Tailwind CSS](https://tailwindcss.com) + [shadcn/ui](https://ui.shadcn.com)
- **Blog** — MDX with syntax highlighting via [rehype-pretty-code](https://rehype-pretty.pages.dev)
- **Animations** — [Magic UI](https://magicui.design)
- **Deployment** — [Vercel](https://vercel.com)

## Running Locally

```bash
npm install
npm run dev
```
